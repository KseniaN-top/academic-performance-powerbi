# Academic Performance Analysis — Power BI Dashboard

Interactive Power BI dashboard analyzing student academic performance across classes, time periods, and key metrics. Built as part of a Data Analytics course (GoIT).

---

## Project Overview

The dashboard provides a multi-page view of academic results, enabling educators or analysts to monitor performance trends, compare classes, and identify areas needing attention.

---

## Dashboard Pages

### 1. Academic Performance Overview
High-level summary with key KPIs and overall trends:
- 4 KPI cards (total metrics at a glance)
- Column chart — performance by category
- Donut chart — distribution breakdown
- Line chart — performance trend over time
- Map — geographic distribution

### 2. Class Performance
Detailed class-level analysis:
- Summary KPI cards
- Data table with class-level results
- Combo chart (columns + line) — combined metrics view
- Clustered column chart — class comparison
- Slicers for filtering by period, class, and other dimensions

### 3. Class Details
Drill-down view:
- Pivot table with granular class data
- Navigation between report pages

### 4. MoM% / AG Details
Month-over-month and above/below average analysis:
- Pivot table with period-over-period changes

### 5. Analytics & Time Intelligence
Advanced time-based analysis:
- 3 line charts — trends over time
- Bar chart — comparative analysis
- Slicers for dynamic filtering

---

## Key Features

- **Time intelligence** — month-over-month comparisons and trend analysis
- **Interactive filtering** — slicers for period, class, and other dimensions
- **Multi-page navigation** — structured report with page navigator
- **Multiple visual types** — cards, line charts, column charts, donut, map, pivot tables
